
Project: Amazon Web Service (AWS) High Availability Website 

Purpose: Proof of concept of deploying a high-availability website with an
Autoscaling Group and Application Load Balancer. The project leverages terraform to deploy and manage the website. 

Variables are used to simplify management of the website deployment.

Consists of 3 public and 3 private subnets in 3 availability zones in 1 AWS region.

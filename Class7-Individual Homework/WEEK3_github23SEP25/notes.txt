Command line commands

1. Touch = add file
2. Rm = delete fie
3. Ls - = list files and folders
4. Pwd - present working directory
5. Cd - change directory(folder)
6. Mkdir - make directory(folder)

Create a new repository from the command line...

echo "# 9-23-25class7" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Lew228/9-23-25class7.git
git push -u origin main

Push an existing repository from the command line....

git remote add origin https://github.com/Lew228/9-23-25class7.git
git branch -M main
git push -u origin main

Git Setup

1.Set your name:
	git config --global user.name "Your Full Name"
2.Set your email:
	git config --global user.email "your.email@example.com"
3. Verify Configuration:
	git config --global --list


To see the status of the files in relation to Github:
	git status

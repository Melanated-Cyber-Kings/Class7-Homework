| ------- | -------------------------------------------- | ------------------- |

| `rm`    | Delete a file                                | `rm oldfile.txt`    |
| `ls`    | List files and folders                       | `ls`                |

| `cd`    | Change directory (folder)                    | `cd Documents`      |
| `mkdir` | Create (make) a new directory (folder)       | `mkdir new_project` |
